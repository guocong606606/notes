# 配置
## dashboard
+ workspace.xml 文件下添加配置
```xml
  <component name="RunDashboard">
    <option name="configurationTypes">
      <set>
        <option value="SpringBootApplicationConfigurationType" />
      </set>
    </option>
  </component>
```
 添加后项目启动自动弹出dashboard
